# Road Accident Data Analysis using Power BI

## Project Overview
This project analyzes road accident data using Power BI and presents insights through interactive dashboards.

## Objectives
- Analyze accident trends
- Identify accident severity levels
- Understand vehicle involvement in accidents
- Visualize accident data using dashboards

## Tools Used
- Microsoft Power BI
- Data Analytics
- Data Visualization

## Dashboard Features
- Total accident statistics
- Accident severity analysis
- Vehicle type analysis
- Monthly accident trends

- Power BI Dashboard (.pbix)
- Dataset
- Project Presentation
- Project Report
